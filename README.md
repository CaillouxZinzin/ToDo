# ToDo
To-Do List application with API (React & Django)

## Frontend
At project root, run :

```cpp=
cd frontend
npm install
npm start
```
```cpp=
cd frontend
yarn install
yarn start
```

Go at http://localhost:3000/

## Backend
At project root, run :

```cpp=
cd backend
python manage.py runserver
```

Go at http://localhost:8000/api/todos/
